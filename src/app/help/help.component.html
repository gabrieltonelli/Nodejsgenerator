Nestjs generator

Install NodeJs

This link for that goto to node web page

Install Nestjs

$ npm i -g @nestjs/cli

Create a project

$nest new project-name

cd project-name

Install librarys

$npm i --save @nestjs/jwt

$npm install bcrypt

$npm install --save nest-winston winston

Libraries for upload files Multer

$npm install @nestjs/platform-express --save

$npm install @types/express -D

Install the type orm library change the driver if you use other database

$npm install --save @nestjs/typeorm typeorm mysql2

example if you use postgres database

$npm install --save @nestjs/typeorm typeorm pg

or if you use microsoft sql server install

$npm install --save @nestjs/typeorm typeorm mssql
